# 首次使用

完成安装和配置后，本文档帮助你开始使用 ChatAI 插件。

## 开始对话

### 使用触发前缀

默认触发前缀为 `#chat`：

```
#chat 你好，请介绍一下自己
```

### @机器人

在群聊中可以 @机器人触发：

```
@机器人 今天天气怎么样？
```

### 多轮对话

AI 会记住对话上下文，支持多轮对话：

```
用户: #chat 我叫小明
AI: 你好小明！很高兴认识你...

用户: #chat 还记得我叫什么吗？
AI: 当然记得，你是小明！
```

## 结束对话

当需要开始新话题时，可以结束当前对话：

```
#结束对话
```

这会清除当前的对话上下文。

## 查看状态

查看插件运行状态：

```
#ai状态
```

输出包含：
- 当前使用的渠道和模型
- 对话统计信息
- 工具加载状态
- 内存使用情况

## 使用预设

### 切换预设

```
#切换预设 猫娘
```

### 查看可用预设

```
#预设列表
```

## 使用工具

如果启用了工具调用，AI 可以使用工具完成任务：

```
用户: #chat 现在几点了？
AI: [调用 get_time 工具]
    现在是 2024年12月15日 14:30:25
```

### 查看可用工具

```
#工具列表
```

## 图片理解

支持发送图片让 AI 分析：

```
[发送图片]
#chat 这张图片里有什么？
```

::: tip 提示
需要使用支持视觉的模型，如 GPT-4V、Claude 3、Gemini Pro Vision
:::

## 引用消息

回复一条消息并触发对话，AI 会理解引用内容：

```
[回复某条消息]
#chat 帮我总结一下这段话
```

## 群聊设置

### 获取群管理面板

```
#群管理面板
```

### 群级配置

每个群可以独立配置：
- 触发方式
- 使用的预设
- 工具权限
- 随机触发概率

## 常用命令速查

| 命令 | 说明 |
|------|------|
| `#chat <消息>` | 发送对话 |
| `#结束对话` | 结束当前对话 |
| `#清除记忆` | 清除长期记忆 |
| `#ai状态` | 查看运行状态 |
| `#ai管理面板` | 获取管理面板 |
| `#群管理面板` | 获取群设置面板 |
| `#预设列表` | 查看可用预设 |
| `#切换预设 <名称>` | 切换预设 |
| `#工具列表` | 查看可用工具 |
| `#ai调试开启` | 开启调试模式 |
| `#ai调试关闭` | 关闭调试模式 |

## 调试模式

遇到问题时，可以开启调试模式查看详细日志：

```
#ai调试开启
```

调试模式会输出：
- API 请求和响应
- 工具调用详情
- 错误堆栈信息

排查完成后记得关闭：

```
#ai调试关闭
```
